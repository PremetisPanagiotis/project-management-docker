## Requirements
- Docker
- Docker Compose

## Run locally
```bash
git clone https://github.com/PremetisPanagiotis/project-management-docker.git
```
```bash
cd project-management-docker
```
```bash
docker compose up
```

## Urls
- Angular: http://localhost:4200
- Java Play: http://localhost:9000
- MySQL: http://localhost:33060
